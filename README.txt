How to run:
1.Inside the APIApp.FrontEnd folder there is an APIApp.sln file. Right click on it and open it using Visual Studio 2019 or later. 
2.Set APIApp.FrontEnd as the Startup project. Then Build the main solution called 'APIApp'. Right click on it and select 'Rebuild solution'.
3.Run the application.
4.To view the APIs and test them, swagger has been installed. This can be selected from the top menu by clicking on the top menu item titled 'Swagger Web API'.
