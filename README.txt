How to run:
1.Inside the APIApp.FrontEnd folder there is an APIApp.sln file. Double click it and it should open the solution. 
2.Right click the APIApp.FrontEnd project in Visual Studio and click  'Set as Startup project'. This should be the startup proejct by default but if not, you will need to perform this step. Then build the main solution called 'APIApp'by right clicking on it and select 'Rebuild solution'.
3.Run the application.
4.To view the APIs and test them, swagger has been installed.This can be selected from the top menu by clicking on the top menu item titled 'Swagger Web API'.
